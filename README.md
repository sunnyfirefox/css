
# 选择器
## CSS元素选择器  
以标签名命名  
 html { color: black; }  
 
 a { font-size : 20px; }  

## CSS 选择器分组  
  一组选择器设置相同的样式  
  
  h2, p {color:gray;}

## CSS 类选择器  
  &lt;div class="cls"&gt;123&lt;/div&gt;  
  
  &lt;p class="cls"&gt;456&lt;/p&gt;  
  
  .cls { color:black; }  
  
  *.cls { color:black; }  
  
  p.cls { color:red; }
  
  .cls,.cls1
  .cls.cls1 区别
  class="cls cls1 cls2"

## CSS ID 选择器  
  ID选择器是唯一的，因为一个文档中，同一个id只会出现一次  
  
  #mostImportant {color:red; background:yellow;}  
  
  &lt;h1 id="mostImportant"&gt;This is important!&lt;/h1&gt;
  

## CSS 属性选择器  
  对标签对应拥有该属性的元素生效  
  
  img[alt] {border: 5px solid red;}  
  

## CSS 后代选择器  
  有父子关系的标签，通常用于一整块组件等情况。  
  
  div h1 {color:red;}    
  &lt;div&gt;  
    
    &lt;h1&gt;  
      
      content  
        
    &lt;/ht&gt;  
      
  &lt;div&gt;
